# Unit Testing

Unit testing project contains arithmetic operations and file input/output operations for practicing Unit Testing.

## Setup

### Dependencies

* [JUnit](https://junit.org/junit4/) 4.12

### Tests

In order to run the tests you should run `mvn test`.
